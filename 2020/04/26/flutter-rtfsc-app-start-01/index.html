总结

FlutterApplication
  1. 从manifest的文件中获取一些值用于Flutter初始化配置
  2. 提取Assets中的资源
  3. 加载libflutter.so
  4. 设置Vsync回调和刷新的fps
FlutterActivity
  1. 创建代理类实例
  2. 创建FlutterEngine实例并进行相关的初始化操作，如绑定所有的pubspec.yaml中配置的第三方plugins等。
  3. 初始化app的背景、状态栏等
  4. 创建FlutterView
  5. 创建FlutterSplashView用于首帧渲染前显示
  6. 配置initRoute，并开始执行Dart代码。
